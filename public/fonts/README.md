# NEX101 Fonts Directory

## Sretch Pro Font

The **Sretch Pro** font is used exclusively for the NEX101 wordmark logo.

### Installation Required

Currently, the Sretch Pro font file is not included in this repository. To add it:

1. Obtain the Sretch Pro font file (`.woff2` or `.woff` format preferred)
2. Place the font file in this directory as:
   - `sretch-pro.woff2` (recommended) or
   - `sretch-pro.woff`

### Current Fallback

Until the actual font is added, the system will fall back to:
- Impact (system font)
- Generic sans-serif

### Notes

- **Figtree** and **Unbounded** (Monumental alternative) are loaded from Google Fonts automatically
- Only Sretch Pro requires manual installation as a local font
